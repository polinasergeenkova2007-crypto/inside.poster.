# INSIDE poster

Интерактивный веб-постер с несколькими мини-играми в пиксельной эстетике.

## Что входит в проект

- первая игра с героем и монстром
- лабиринт с огнём и выходом
- дорожная мини-игра
- игра `4x4` по механике `match-3`
- игра с бурундуками
- игра с лодкой
- игра со сканированием

## Технологии

- `HTML`
- `CSS`
- `JavaScript`

## Использование ИИ

Использование ИИ в проекте было минимальным. `Codex` применялся только как вспомогательный инструмент для проработки отдельных игровых механик и некоторых технических решений.

## Роль автора

Автор проекта самостоятельно:

- определял визуальную концепцию
- выбирал игровые механики
- принимал решения по структуре интерфейса
- проверял и дорабатывал предложенный код
- вручную настраивал композицию, расположение элементов и финальный вид проекта
