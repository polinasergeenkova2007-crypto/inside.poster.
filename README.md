# INSIDE poster

Интерактивный веб-постер с несколькими мини-играми в пиксельной эстетике.

## Что входит в проект

- первая игра с героем и монстром
- лабиринт с огнём и выходом
- дорожная мини-игра
- игра `4x4` по механике `match-3`
- игра с бурундуками
- игра с лодкой
- игра со сканированием

## Технологии

- `HTML`
- `CSS`
- `JavaScript`

## Как использовалась помощь ИИ

При разработке проекта использовалась помощь ИИ `Codex`. ИИ применялся как инструмент программной поддержки и ускорения разработки, а не как замена авторской работы.

С помощью ИИ были:

- структурированы отдельные игровые модули и их состояния в `JavaScript`
- собраны покадровые анимации через `requestAnimationFrame`
- настроены игровые тайминги через `setTimeout` и `setInterval`
- реализованы CSS-состояния персонажей и игровых объектов, например `is-jumping`, `is-attacking`, `is-burning`, `is-complete`
- настроены перемещения объектов через CSS-переменные, обновляемые из `JavaScript`
- реализованы проверки столкновений через `getBoundingClientRect()`
- описаны hitbox-зоны для опасных объектов в лабиринте
- собрана логика игры `4x4`: выбор соседних клеток, обмен, поиск совпадений, удаление совпадений и дозаполнение поля
- реализована процедурная сборка кирпичного пути на основе `SVG polyline`
- добавлена поддержка разных способов ввода: клавиатура, мышь, `pointer`, `touch`
- настроено управление служебными состояниями игры: сброс, рестарт, очистка таймеров, завершение раунда

## Для чего использовался ИИ

ИИ использовался для следующих задач:

- ускорение написания и переработки кода
- помощь в проектировании игровой логики
- упрощение отладки анимаций и состояний
- помощь в адаптации интерфейса под разные экраны
- поиск более удобных решений для интерактивных механик
